Volquetes POOmBA

Somos el grupo POOmBA, integrado por: Mónica Nieto, Lucas Marques de Paiva, Federico Balbuena y Gustavo Zammataro.

Como proyecto final integral de las materias Frontend y Backend, realizamos un ABM o CRUD (por sus siglas en ingles), para un sistema de gestión de volquetes, el objetivo es poder crear órdenes de servicio, poder editarlas, revisarlas o eliminarlas según corresponda. Para realizar este proyecto utilizamos: FrontEnd: HTML, CSS, JavaScript, Angular, Bootswatch. BackEnd: TypeScript, Node, Express, MongoDB. Para gestión del proyecto, utilizamos un sistema de versionado con git y subimos el proyecto a un repositorio en GitHub, donde trabajamos en forma colaborativa dividiendo tareas y validando criterios entre los miembros del equipo. Para la gestión de las tareas empleamos Trello, donde fuimos registrando el progreso y el estado de las tareas.
